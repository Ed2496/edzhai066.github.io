## 🛠️ 使用與部署

本專案為 GitHub Pages 靜態網頁，部署步驟如下：

1. 本 repo 須為 Public
2. 前往 Settings → Pages
3. 選擇「Source: `main` branch」→ 點選 Save
4. 幾分鐘後網站即可上線：`https://<your-username>.github.io`
